Demo of Angular Change detection

Selectable Changedetection strategy animated.

How it works

There are 2 types of ChangeDetection strategy:

  • Default: Every event triggers the ChangeDetection.
  • OnPush: Only triggers when an component or children component event handler gets triggered or an template observable emits

The projects shows the detection by interpolating a method in template.

{{'{{'}} test() {{'}'}}{{'}'}}

The function gets called on every cycle, because the framework does not know what is the return value.

Method-call triggers an animation which changes the background color for 500ms.

The strategy changing mechanism based on a global app state by NgRx store.

Each component has a boolean which can be changed by the buttons.
